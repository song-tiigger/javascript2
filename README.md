# javascript2
뽀모도로 타이머

## 프로젝트 소개
25분간 집중해서 일한 다음 5분간 휴식하는 방식의 뽀모도로 기법을 타이머에 적용했습니다. <br>
집중 시간인 25분이 끝나면 자동으로 휴식시간으로 전환됩니다.

## 사용 기술
- html5/ scss
- jQuery/ javascript

## 새롭게 알게 된 것
- 숫자 앞에 0을 추가하는 코드
- setInterval 함수가 적용된 버튼을 계속 누르면 중첩되어 적용됨 그래서 중복하여 클릭할 수 없게 처리해야 함
